Task Manager API

Esta é a API do gerenciador de tarefas. Abaixo estão as funções Cloud disponíveis:

GET getTasks

Retorna todas as tarefas cadastradas.

GET getTaskById

Retorna uma tarefa específica pelo ID.

Parâmetros: taskId (string)

POST createTask

Cria uma nova tarefa.

Parâmetros: title (string), description (string), status (string)

PUT updateTask

Atualiza uma tarefa existente.

Parâmetros: taskId (string), title (string, opcional), description (string, opcional), status (string, opcional)

DELETE deleteTask

Exclui uma tarefa.

Parâmetros: taskId (string)

GET getTasksByStatus

Filtra tarefas por status.

Parâmetros: status (string)

Esta API foi desenvolvida usando o Back4App e Parse Server.
